# echotext

EchoText is a compact assistive tool designed to read such information aloud using OCR and Text-to-Speech (TTS). The system simplifies daily activities, enhances independence, and reduces reliance on others for product-related information.

Introduction
Many visually challenged individuals struggle with everyday tasks that rely heavily on reading printed information. Critical details like expiry dates on food items, nutritional values, and medicine labels are often printed in small fonts and inconsistent formats, making them extremely difficult to access without assistance.
